A comprehensive knowledge management system for organizing, storing, and retrieving information efficiently.

## Features

- **Article Management**: Create and organize knowledge articles with rich text content
- **Category System**: Organize articles into hierarchical categories
- **Search Functionality**: Full-text search across all articles
- **Tagging System**: Tag articles for better organization and discovery
- **User Management**: Role-based access control for content management

## Tech Stack

- **Backend**: FastAPI with SQLAlchemy
- **Frontend**: Vanilla JavaScript with modern CSS
- **Database**: SQLite
- **Deployment**: Docker & Docker Compose

## Quick Start

1. **Navigate to the app**:
   ```bash
   cd Knowledge-Base
   ```

2. **Start the application**:
   ```bash
   docker compose up -d
   ```

3. **Open your browser**:
   Navigate to `http://localhost:8000`

## Usage

- Create categories to organize your knowledge
- Add articles with rich content
- Use tags to make articles easily discoverable
- Search through your knowledge base quickly
